Quick Start Guide - Forgot Password FeatureÔÉÅ

What‚Äôs New?ÔÉÅ

Users and owners can now reset their forgotten passwords using OTP (One-Time Password) verification.

How to Use (User Guide)ÔÉÅ

For Users/Students:ÔÉÅ

  1. Go to Student Login page (/login-student)

  2. Click ‚ÄúForgot your password?‚Äù link

  3. Enter your registered email address

  4. Check your email for the OTP (6-digit code)

  5. Enter the OTP on the verification screen

  6. Create a new password (minimum 8 characters)

  7. Log in with your new password

For Owners:ÔÉÅ

  1. Go to Owner Login page (/login-owner)

  2. Click ‚ÄúForgot your password?‚Äù link

  3. Follow the same process as above

Technical DetailsÔÉÅ

New Endpoints:ÔÉÅ

  • GET/POST /forgot-password - Main forgot password page

  • POST /send-otp - Send OTP to email

  • POST /verify-otp - Verify OTP code

  • POST /reset-password - Reset password

Request/Response Examples:ÔÉÅ

1. Send OTP:

POST /send-otp
{
  "email": "user@example.com"
}

Response:
{
  "success": true,
  "message": "OTP sent successfully to your email"
}

2. Verify OTP:

POST /verify-otp
{
  "email": "user@example.com",
  "otp": "123456"
}

Response:
{
  "success": true,
  "message": "OTP verified successfully"
}

3. Reset Password:

POST /reset-password
{
  "email": "user@example.com",
  "otp": "123456",
  "new_password": "NewPassword@123"
}

Response:
{
  "success": true,
  "message": "Password reset successfully. Please log in with your new password."
}

FeaturesÔÉÅ

‚úÖ 6-digit OTP sent to registered email
‚úÖ 10-minute OTP validity - OTP expires after 10 minutes
‚úÖ Resend OTP - Users can request a new OTP (60-second cooldown)
‚úÖ Password Strength Indicator - Real-time feedback while typing
‚úÖ Email Change Option - Users can go back and use different email
‚úÖ Security Checks - Multiple OTP validations during reset process
‚úÖ Responsive Design - Works on mobile and desktop
‚úÖ Professional Email - Branded HTML email with OTP

SecurityÔÉÅ

  • OTPs are 6-digit random numbers

  • Each OTP is valid for only 10 minutes

  • OTP is deleted immediately after successful password reset

  • Passwords are hashed with bcrypt before storage

  • Email verification ensures only account owner can reset password

  • No indication whether email exists (privacy-friendly error messages)

TroubleshootingÔÉÅ

OTP not received in email?ÔÉÅ

  • Check spam/junk folder

  • Verify email configuration in .env file

  • Check application logs for email sending errors

‚ÄúOTP has expired‚Äù error?ÔÉÅ

  • OTP is only valid for 10 minutes

  • Click ‚ÄúResend OTP‚Äù link to request a new one

‚ÄúInvalid OTP‚Äù error?ÔÉÅ

  • Make sure you entered the correct 6-digit code

  • No spaces before/after OTP

  • Each OTP can only be used once

Password validation errors?ÔÉÅ

  • Password must be at least 8 characters long

  • Try adding numbers, uppercase, lowercase, and special characters

  • Ensure both password fields match exactly
